# Parcial2
Este código permite calcular el salario de ciertos trabajadores tras aplicarle una subida porcentual al que tenían anteriormente.
Para esto se solicita por consola el porcentaje de subida a aplicar, se calcula y se suma al salario base, tras esto se muestra por mensaje el nombre, cargo y salario final de cada trabajador.
Los tests realizados constan de introducir datos invalidos o negativos.
El código se encuetra actualmente en su versión 1.1, así que todo feedback será bienvenido.
